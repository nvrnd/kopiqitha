🔒 Keamanan Aplikasi ini:
✅ Password disimpan terenkripsi di Firebase Authentication — tidak bisa dibaca siapapun termasuk developer.
✅ Data toko dilindungi Firestore Security Rules — hanya user yang login yang bisa akses.
✅ Tidak ada password di kode HTML — akun dikelola penuh di Firebase Console.
✅ Login persisten aman via Firebase session token yang dienkripsi browser.

📋 Cara Admin tambah/hapus user:
Buka Firebase Console → Authentication → Users → Add user / Delete user.
